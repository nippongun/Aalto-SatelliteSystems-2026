**Module-level constants:**
```python
ALTITUDE_CONTEXT = re.compile(
    r"(?:orbit|altitude|LEO|apogee|perigee).*?(\d+(?:\.\d+)?)\s*km"
    r"|(\d+(?:\.\d+)?)\s*km.*?(?:orbit|altitude|LEO|apogee|perigee)",
    re.IGNORECASE
)
UNIT_PATTERN = re.compile(
    r"\d+(?:\.\d+)?\s*(?:km|m|dB|W|kg|bps|MHz|GHz|°|deg|%|ms|s\b)"
)
AI_HEADING = re.compile(r"^#+\s+AI\s+usage", re.IGNORECASE | re.MULTILINE)
```

**CheckResult dataclass:**
```python
@dataclasses.dataclass
class CheckResult:
    name: str
    status: str   # "PASS", "FAIL", or "WARNING"
    detail: str
    line_refs: list  # list of (filename_str, line_no_int) tuples
```

**Seven check functions** (all accept explicit Path/dict params, never open files via hardcoded paths):

1. `check_altitude_consistent(content_dir: Path, config: dict) -> CheckResult`
   - Scan each *.md file line by line; for each line that matches ALTITUDE_CONTEXT, extract the km value from group(1) or group(2) (whichever matched); accumulate {value: [(file, lineno)]}
   - If len(unique values) > 1: FAIL with sorted unique values and all refs
   - Else: PASS

2. `check_mass_budget(config: dict, budget_path: Path) -> CheckResult`
   - If not budget_path.exists(): return PASS with "budgets/mass_budget.csv not found — skipped"
   - Use csv.DictReader; for each row try float(row["mass_kg"]) except (ValueError, KeyError): skip
   - If total > config.get("mass_kg", 0): FAIL with total and limit
   - Else: PASS

3. `check_power_budget(config: dict, budget_path: Path) -> CheckResult`
   - If "eps_power_W" not in config: return CheckResult("Power budget", "WARNING", "eps_power_W not defined in mission_config.yaml — add it before this check is meaningful", [])
   - If not budget_path.exists(): return PASS with skip note
   - Use csv.DictReader; sum "power_W" column; compare to config["eps_power_W"]
   - If total > eps_power: FAIL; else PASS

4. `check_requirement_units(req_data: dict) -> CheckResult`
   - Iterate objectives → requirements; if not UNIT_PATTERN.search(req["text"]): add req["id"] to missing list
   - If missing: FAIL listing IDs; else PASS

5. `check_objective_coverage(req_data: dict) -> CheckResult`
   - For each objective: if not obj.get("requirements"): add obj["id"] to orphaned list
   - If orphaned: FAIL; else PASS

6. `check_success_criteria(content_dir: Path) -> CheckResult`
   - Read content_dir / "01_motivation.md"; if not exists: FAIL "01_motivation.md not found"
   - Check text.lower() for "minimal success" and "full success"; report missing keywords
   - If any missing: FAIL; else PASS

7. `check_ai_usage_section(content_dir: Path) -> CheckResult`
   - For each *.md in content_dir: if AI_HEADING.search(text): return PASS with filename
   - If no file found: FAIL

**`run_all_checks(content_dir, config, req_data, mass_budget_path, power_budget_path)`** calls all seven in sequence, returns list of CheckResult.

**`write_report(results, output_path)`** writes Markdown with "## PASS: name" or "## FAIL: name" or "## WARNING: name" headers, detail line, and bullet refs.

**`main()`** uses argparse. Positional optional `content_dir` arg (nargs="?"), plus `--config`, `--requirements`, `--output`. Derives mass_budget_path and power_budget_path from `repo_root / "budgets" / "mass_budget.csv"` etc. (passed as args if needed, or derived via repo_root = Path(__file__).parent.parent). Always writes report; exits 1 if any FAIL (WARNING does not trigger exit 1).
